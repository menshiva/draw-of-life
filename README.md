## Requirements 🧬

*  C11 compiler
*  [CMake](https://cmake.org/)
*  [SDL2](https://www.libsdl.org/)

## Build and Run 🚀

### Install SDL2

```bash
sudo apt install libsdl2-dev
```

### Build and run application

```bash
mkdir release
cd release
cmake .. && make
./DrawOfLife
```
